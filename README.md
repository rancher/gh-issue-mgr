# gh-issue-mgr

A collection of GitHub Actions to help manage GitHub issues and projects efficiently. This repository contains various actions that automate common issue management tasks.

## Available Actions

### [Move to Next Iteration](move-to-next-iteration/README.md)
Automatically moves issues to the next iteration in GitHub Projects. This action helps streamline project management by automatically transitioning issues based on their current status.
